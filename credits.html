About Exoix

Exoix is a Discord status tracker that shows real-time information about Discord users, including their current status, Spotify activity, gaming sessions, and custom data.

Features:

  • Real-time Discord status tracking
  • Spotify integration with album art
  • Game activity detection
  • Custom user data (KV store)
  • Responsive design
  • Auto-refresh functionality
